# Web Game Project

##  Description

This project is a web-based game developed using HTML, CSS, and JavaScript.  
It includes a simple authentication system and a basic local data structure to simulate user storage.

---

##  How it works

- The project runs directly in the browser.
- Simply open `index.html` to start.

---

##  Authentication System

Users must create an account before playing.

Validation rules:
- Valid email format
- Password with at least 8 characters

User data is stored locally using a simple array (no database).

---

##  Gameplay Access

- After creating an account, users can log in.
- Only logged-in users can access the game.
- If a user tries to play without logging in, a warning message is displayed.

---

##  Design

The project includes a PDF inside the `docs/` folder showing the UI layout and design structure used during development.
